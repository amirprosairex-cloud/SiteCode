� EmailJS — отправка кодов 2FA и сброса пароля
EmailJS позволяет отправлять письма прямо из браузера без сервера.
1. Зарегистрируйтесь на emailjs.com (бесплатно — 200 писем/месяц)
2. Создайте Email Service (Gmail, Outlook, SMTP и др.)
3. Создайте Email Template — используйте переменные: {{otp_code}}, {{to_email}}, {{purpose}}
4. Вставьте Public Key, Service ID и Template ID ниже
Account → API Keys → Public Key
💡 До настройки EmailJS коды будут показываться в тосте прямо на экране (демо-режим). 2FA и сброс пароля работают и без настройки — просто код виден на экране, а не в письме.
